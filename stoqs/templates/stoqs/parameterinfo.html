Activities in {{ dbAlias }}

This page provides an overview of the data that has been loaded into the {{ dbAlias }} database. There are sample links in the third column that demonstrate how to get data out of the database. The way to use them is as follows:
  • Click the 'count' link for a parameter
  • Adjust the start/end times and start/end depths in the url to get data you want and re-issue the count request
  • See that the count is about a 1,000 points (if it's more than this then the .kml request will take more than 30 seconds)
  • You can add a stride/10 after the depth criteria to take every 10th point so that the count is reduced
  • Change 'count' to 'data.html' or 'data.kml' to see the data. You can use 'data.csv' to load the data into other software
  • The perf.kml request is used to understand where the server is spending time in order to serve a request
